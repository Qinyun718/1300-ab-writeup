Time to Completion:

Null Hypothesis: the time it takes to complete the desired task on version A is equal to that of version B

Alternative Hypothesis: the time it takes to complete the desired task on version A is less than that of version B

Return Rate:

Null Hypothesis: the number of times a user returned to the page on version A is equal to that of version B

Alternative Hypothesis: the number of times a user returned to the page on version A is less than that of version B

Reasoning behind alternative hypothesis:

Since I changed the layout from the vertical view to the horizontal view, it would provide more convenience for audience to add items and view the prices. Thus, both of the time to completion and return rate should be smaller than the original version.

Data Collection Process

The data collection process was completed during the studio time. With the deployed Heroku link, participants tested on either version A or version B with the purpose of filling $150 shopping cart. Their records would be used to calculate different metrics for the AB testing process.

.03 Results

Broad Takeaways

  • The sample size is too small to obtain significant results.
  • The p-value for the t-test is approaching to the significance level, indicating that the horizontal layout might decrease the time to completion.
  • Since the data collection part was not done with natural completion, the results might include some bias.
